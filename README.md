# Digontom Future Kids 🚀

A modern, responsive web application designed to equip kids with real-world tech skills, helping them become confident and future-ready. This project features a premium, AI-company aesthetic inspired by Digontom, utilizing vibrant orange/amber gradients and a sleek dark mode layout.

## 🌟 Features

- **Premium UI/UX:** A stunning dark-themed interface with glowing effects and dynamic gradients.
- **Responsive Design:** Fully optimized for both mobile and desktop screens.
- **Modern Tech Stack:** Built with React, Vite, and Tailwind CSS for blazing fast performance.
- **Engaging Hero Section:** An eye-catching landing area designed to build trust and encourage enrollment.

## 🛠️ Tech Stack

- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** React Icons (or equivalent)
- **Routing:** React Router (if applicable)

## 🚀 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Digontom-Future-Kids
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

### Running Locally

Start the development server with Hot Module Replacement (HMR):

```bash
npm run dev
```

The application will be available at `http://localhost:5173/` (or whichever port Vite provides).

### Building for Production

To create an optimized production build:

```bash
npm run build
```

You can preview the production build using:

```bash
npm run preview
```

## 🎨 Theme Details

The styling reflects a tech-forward visual language:
- **Primary Colors:** Black background (`#0a0a0a`), White text.
- **Accents:** Gradients transitioning from Orange (`#ea580c`) to Amber (`#f59e0b`).
- **Effects:** Soft glowing shadows, screen blend modes, and backdrop blurs to create depth.
